# 数据预处理与标准化

> 提取自源文档第三章。本章定义所有后续合成分析（见 [synthesis_methods.md](synthesis_methods.md)）的前置数据规范，被全部三个场景模块依赖。

## 1. 年龄-深度建模

年龄不确定性是古生态学 meta 分析区别于其他领域的核心特征。所有后续合成分析的前提是获得每个岩芯的**年龄集合**（age ensemble），而非单一年龄-深度曲线。

- **BAM（Banded Age Model）**：Comboul 等人 2014 年提出，Kaufman 2020 验证其与 Bacon 的年龄估计可比——BAM 的 RMSE 为 **251 年**，Bacon 为 **198 年**，差异在可接受范围内。BAM 的优势在于纯 Python 实现，不依赖 R 环境。当用户 R 环境被阻断时，BAM 是首选方案。
- **Bacon / Clam**：须通过 rpy2 桥接 R 环境。若已有 Bacon 年表（如用户已于 7 月 17 日完成各研究站点的 Bacon 模型），可直接消费其年龄集合输出进行后续分析，无需重新建模。年龄集合中的每个成员本身就是一条单调的年龄-深度曲线，可直接作为蒙特卡洛集合的一个实现。

> 注意：年龄扰动须保持地层顺序——不可从年龄后验分布中独立采样每个深度的年龄，否则破坏深度-年龄单调关系。正确做法是从 BAM/Bacon 输出的完整年龄集合成员中整体采样（详见 [synthesis_methods.md](synthesis_methods.md) 第 5 节）。

**工具选择**：BAM 纯 Python 实现（见 [python_toolchain.md](python_toolchain.md) 第 1 节）；Bacon/Clam 须 rpy2 桥接，当 R 环境被阻断时推荐 BAM 替代。

## 2. z-score 标准化

Izdebski 2022 和 Power 2008 均采用 z-score 标准化消除不同岩芯间的量纲差异。对每个岩芯的每个分类群，以研究时段的均值和标准差计算：

```
z = (x - μ) / σ
```

其中 μ、σ 分别为该分类群在整个研究时段的均值和标准差。此操作将所有记录转换为相对于各自基线的偏差，使不同点位、不同分类群的变化幅度具有可比性。

**基准时段选择影响结果解读**：Izdebski 2022 使用整个研究时段作为基准，"高于均值/低于均值"是相对于整个研究窗口而非现代值。若研究目标是评估相对于现代基线的变化，须调整基准时段。

**实现要点**：z-score 标准化可由 scikit-learn / pandas 原生完成（见 [python_toolchain.md](python_toolchain.md) 第 1 节），无需额外安装。对每个岩芯的每个分类群独立计算 μ 和 σ，不可跨岩芯混用统计量。

## 3. 代用指标特定预处理

不同代用指标需要特定的预处理步骤：

### 3.1 花粉数据
- **分类群命名统一**：不同实验室使用不同的分类群命名体系，须建立映射表统一到一致分类框架。这是最基础的步骤。
- **多环境保存偏倚（含喀斯特/干旱/热带/湖泊/海洋预设）**：不同沉积环境可能导致分类群保存存在系统性偏倚——以喀斯特碱性土壤为例，喜酸分类群（如杜鹃花科）可能被低估，耐碱分类群（如禾本科）可能被相对高估；干旱、热带、湖泊、海洋等环境各有其保存偏倚模式。此偏倚须在预处理中显式记录，并在结果解读时纳入考虑。可通过对保存敏感分类群（如杜鹃花科）和保存耐受分类群（如禾本科）分别分析，评估保存偏倚对结论的影响，作为场景二和场景三的敏感性检验之一。

### 3.2 炭屑数据
Power 2008 使用 Global Charcoal Database (GCD) 的标准化炭屑通量。不同岩芯的炭屑通量单位可能不同（颗粒/cm²/年 vs 体积浓度），须统一转换后再合成。

### 3.3 温度代理（brGDGTs、δDwax）
须先完成校准模型拟合，将分子标志物浓度转换为温度/水文指标，再进行 z-score 标准化。校准模型的不确定性须作为独立误差层在后续集合传播中纳入（见 [validation.md](validation.md) 第 2 节）。

## 4. 时空对齐

### 4.1 空间对齐
- **等面积网格化**（Kaufman 2020）：typically 1°×1° 或更精细。
- **地理区域聚类**（Izdebski 2022）：对欧洲以约 200km 半径划分空间单元。

**目标区域推荐聚类方案**：该区域沉积点位密度不足以支持精细网格化，且复杂地貌的空间异质性（如洼地、谷地、坡地）使地理聚类比经纬度网格更有生态意义。

### 4.2 时间对齐
将不同分辨率的岩芯重采样到统一时间网格。常用方案是 10 年或 50 年等间距插值。**插值时须使用年龄集合中的每个成员独立插值**，以传播年龄不确定性。

**关键操作**：对年龄集合中的每个成员（每条单调的年龄-深度曲线）独立执行重采样到统一时间网格，再传递给下游的蒙特卡洛集合（见 [synthesis_methods.md](synthesis_methods.md) 第 5 节）。不同记录的时间分辨率差异须先重采样到统一网格再合成，否则高频记录会主导合成结果。

## 5. 常见陷阱汇总

源文档第三章及相关章节标注的预处理陷阱：

- **年龄独立性陷阱**：不可从年龄后验分布中独立采样每个深度的年龄，须整体采样完整年龄集合成员以保持地层单调性。
- **基准时段陷阱**：z-score 基准时段选择影响"高于/低于均值"的解读，须与研究目标对齐（整个研究窗口 vs 现代基线）。
- **多环境保存偏倚陷阱**：不同沉积环境（如喀斯特碱性土壤、干旱区、热带等）可能导致分类群保存存在系统性偏倚，须显式记录并对保存敏感/耐受分类群分别分析。
- **炭屑单位陷阱**：不同岩芯的炭屑通量单位（颗粒/cm²/年 vs 体积浓度）须统一转换后再合成。
- **时间分辨率陷阱**：不同记录的分辨率差异须先重采样到统一网格再合成，否则高频记录主导合成结果。

## 6. 用户实际数据适配考虑

源文档第 11.3 节针对古生态学与古气候学多代理指标研究给出以下适配建议：

- **已有 Bacon 年表的岩芯**：各研究站点的 Bacon 模型可直接消费年龄集合输出，进行场景二多站点植被变化综合分析。
- **代用指标**：δDwax/brGDGTs 等连续值代理或花粉/硅藻等分类群代理数据可用于场景一代用指标有效性评估。校准模型的 RMSEP 作为不确定性传播中的校准误差层。
- **历史代用指标**：灾害记录、疫病、粮价等历史文献数据可用于场景三跨区域人地归因分析。历史事件年份须从历史文献中独立确定，不可循环使用花粉数据自身确定。
